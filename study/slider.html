Swiper.js를 사용
1. swiper.js 사이트 https://swiperjs.com/
2. 사용 태그 (Default)
(1) swiper-container - main Swiper element where you define all parameters
(2) swiper-slide - Swiper slide element
3.속성
  1. navigation="true" <> 표시
  2. Pagination="true" 하단 페이지 ㅇ 표시 (슬라이드 수만큼의 bullet)
  3. pagination-dynamic-bullets : 활성 bullet이 커지고, 나머지는 작아지거나 사라짐
  4. pagination-type="progressbar" : 상단 진행바
  5. pagination-type="fraction" : n/total
  6. custom : init="false" 후, 자바스크립트로 커스터마이징
  7. scrollbar-hide="true"
  8. direction="vertical" 위아래 슬라이드
  9. pagination-clickable="true" : Pagination 클릭으로 이동
  10. -> 2~ bullet중 하나만 적용
  11. spacebetween : spaceBetween은 클래식 방식에서 적용 https://cdn.jsdelivr.net/npm/swiper@11/swiper-bundle.min.css"
  12. space-between : 컴포넌트 방식에서 적용 https://cdn.jsdelivr.net/npm/swiper@12/swiper-element-bundle.min.js
